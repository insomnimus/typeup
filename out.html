tables

tables are of the form #<delimiter>{} . Delimiters can be multi-character. The first row becomes the table header. Each item is trimmed of its whitespace

custom blocks

custom blocks are of the form function:{input} Different translators will have different built-in functions. To run shell commands, prefix the function with ! . Some translators have a secure mode that forbids these.

metadata

you can specify metadata in a @{} block, as key-value pairs of strings. These can be used by static site generators, Wikis and other programs They need to be of the form key = value . Whitespace is trimmed Additionaly, the shortcut =# exists which creates a h1 element with its argument and sets the title attribute to the content
